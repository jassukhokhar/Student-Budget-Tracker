# 🎓 Student Budget Tracker : https://expense-tracker-2-28z4.onrender.com/home

A *full-stack MERN web application* that helps students efficiently manage their expenses and budgets.  
Built with *Node.js, Express.js, MongoDB (Mongoose), EJS, and TailwindCSS*.  

## ✨ Features
- 💰 Expense categorization (food, travel, bills, etc.)  
- 📊 Budget tracking with data visualization and alerts  
- 🔐 Secure authentication with *Passport.js*  
- 📧 Email verification and OTP-based login/signup  
- 📱 Responsive UI with *EJS + TailwindCSS*  
- 🗄 Data storage using *MongoDB (Mongoose)*  

## 🛠 Tech Stack
- *Frontend:* EJS, TailwindCSS  
- *Backend:* Node.js, Express.js  
- *Database:* MongoDB (Mongoose)  
- *Authentication:* Passport.js, Email + OTP verification
